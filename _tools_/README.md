# Tools

Here are some other tools I need for running my website.

## Shell scripts
- `unews`: updates my RSS feed (using `sfeed`) to produce `feeds.html`
- `pnews`: prettifies `feeds.html` and posts it on my website;
- `cronnews`: sample cronjob for automatically doing the above every 3 hours;

## Programs
- [`sfeed`](https://git.codemadness.org/sfeed/file/README.html) with my content;
