Fie $a, b \in \RR$ »Щi $f : [a, b) \to \RR$ o func»Ыie \emph{local integrabilƒГ}
(i.e.\ integrabilƒГ pe orice interval compact $[u, v] \seq [a, b)$).

\[

integrala se nume»Щte \emph{divergentƒГ}.

improprie (la $\infty$) $\ds\int_a^\infty f(x) dx$ se nume»Щte \emph{convergentƒГ}
dacƒГ limita:

Criteriile de convergen»ЫƒГ pentru integralele improprii s√Ѓnt foarte asemƒГnƒГtoare
cu cele pentru serii (aminti»Ыi-vƒГ, cƒГ, de fapt, integralele definite se construiesc

A»Щadar, avem:

\textbf{Criteriul lui Cauchy} (general): Fie $f:[a, b) \to \RR$ local integrabilƒГ.
Atunci integrala $\ds\int_a^b f(t) dt$ este convergentƒГ dacƒГ »Щi numai dacƒГ:
\[
  \forall \varepsilon > 0, \exists b_\varepsilon \in [a, b) \ai %
  \forall x, y \in (b_\varepsilon, b), \big| \int_x^y f(t) dt \big| < \varepsilon.
\]

\vspace{0.5cm}

\textbf{Criteriul de compara»Ыie} (\qq{termen cu termen}): Fie $f, g : [a, b) \to \RR$
astfel √Ѓnc√Ѓt $0 \leq f \leq g$.
\begin{itemize}
\item DacƒГ $\ds\int_a^b g(x) dx$ este convergentƒГ, atunci »Щi integrala
  $\ds\int_a^b f(x) dx$ este convergentƒГ;
\item DacƒГ integrala $\ds\int_a^b f(x) dx$ este divergentƒГ, atunci »Щi integrala
  $\ds\int_a^b g(x) dx$ este divergentƒГ.
\end{itemize}

\vspace{0.5cm}

\textbf{Criteriul de compara»Ыie la limitƒГ}: Fie $f, g : [a, b) \to [0, \infty)$,
astfel √Ѓnc√Ѓt sƒГ existe limita:
\[
  \ell = \lim_{x \to b} \frac{f(x)}{g(x)}.
\]
\begin{itemize}
\item DacƒГ $\ell \in [0, \infty)$, iar $\ds\int_a^b g(x) dx$ este convergentƒГ,
  atunci $\ds\int_a^b f(x) dx$ este convergentƒГ;
\item DacƒГ $\ell \in (0, \infty)$ sau $\ell = \infty$, iar $\ds\int_a^b g(x) dx$
  este divergentƒГ, atunci »Щi $\ds\int_a^b f(x) dx$ este divergentƒГ.
\end{itemize}

\vspace{0.5cm}

\textbf{Criteriul de compara»Ыie cu $\dfrac{1}{x^\alpha}$}:
Fie $a \in \RR$ »Щi $f : [a, \infty) \to [0, \infty)$ local integrabilƒГ, astfel
√Ѓnc√Ѓt sƒГ existe:
\[
  \ell = \lim_{x \to \infty} x^\alpha f(x).
\]
\begin{itemize}
\item DacƒГ $\alpha > 1$ »Щi $0 \leq \ell < \infty$, atunci $\ds\int_a^\infty f(x) dx$
  este convergentƒГ;
\item DacƒГ $\alpha \leq 1$, iar $0 < \ell \leq \infty$, atunci $\ds\int_a^\infty f(x) dx$
  este divergentƒГ.
\end{itemize}

\vspace{0.5cm}

\textbf{Criteriul de compara»Ыie cu $\dfrac{1}{(b - x)^\alpha}$}: Fie $a < b$ »Щi
$f : [a, b) \to [0, \infty)$, local integrabilƒГ, astfel √Ѓnc√Ѓt sƒГ existe:
\[
  \ell = \lim_{x \to b} (b - x)^\alpha f(x).
\]
\begin{itemize}
\item DacƒГ $\alpha < 1$ »Щi $0 \leq \ell < \infty$, atunci $\ds\int_a^b f(x) dx$
  este convergentƒГ;
\item DacƒГ $\alpha \geq 1$ »Щi $0 < \ell \leq \infty$, atunci $\ds\int_a^b f(x) dx$
  este divergentƒГ.
\end{itemize}

\vspace{0.5cm}

\textbf{Criteriul lui Abel}: Fie $f, g : [a, \infty) \to \RR$, cu proprietƒГ»Ыile:
\begin{itemize}
\item $f$ este de clasƒГ $\kal{C}^1$, $\ds\lim_{x \to \infty} f(x) = 0$, iar
  $\ds\int_a^\infty f'(x) dx$ este absolut convergentƒГ;
\item $g$ este continuƒГ, iar $G(x) = \ds\int_a^x f(t) dt$ este mƒГrginitƒГ pe
  $[a, \infty)$.
\end{itemize}
Atunci integrala $\ds\int_a^\infty f(x) g(x) dx$ este convergentƒГ.

Un caz particular care ne intereseazƒГ este acela al integralelor
improprii cu parametri. Un exemplu este:
\[
    I(m) = \int_0^{\dfrac{\pi}{2}} \ln(\cos^2 x + m^2 \sin^2 x)dx, \quad m > 0,
\]
care se poate rezolva folosind tehnica de derivare √Ѓn interiorul
integralei, adicƒГ:
\[
    I'(m) = \int_0^{\dfrac{\pi}{2}} \dfrac{\partial}{\partial m} %

\]

\textbf{Exemple rezolvate:}

SƒГ se calculeze integralele, folosind derivarea sub integralƒГ:
\begin{enumerate}[(a)]
\item $ I(m) = \ds\int_0^{\frac{\pi}{2}} \ln(\cos^2 x + m^2 \sin^2 x) dx, m > 0 $;
\item $ I(a) = \ds\int_0^{\frac{\pi}{2}} \ln(a^2 - \sin^2 \theta) d\theta, a > 1 $;
\item $ I(a) = \ds\int_0^1 \frac{\arctan ax}{x \sqrt{1 - x^2}} dx, a > 0 $.
\end{enumerate}

\emph{Solu»Ыii:}

(a) DacƒГ considerƒГm func»Ыia:
\[
  f(x, m) = \ln(\cos^2 x + m^2 \sin^2 x),
\]

\[
  \frac{\partial f}{\partial m} = I'(m) = 2m \int_0^{\frac{\pi}{2}} %
  \frac{\sin^2 x}{\cos^2 x + m^2 \sin^2 x} dx
\]

Pentru a calcula integrala, facem schimbarea de variabilƒГ $\tan x = t$ »Щi atunci:
\[
  dt = \frac{1}{\cos^2 x} dx.
\]

Integrala ini»ЫialƒГ se poate prelucra:
\[
  I'(m) = 2m \int_0^{\frac{\pi}{2}} \frac{\sin^2 x}{cos^2 x(1 + m^2 \tan^2 x)} dx.
\]
A»Щadar, pentru a ob»Ыine $\sin^2 x$ √Ѓn func»Ыie de $t$ calculƒГm:
\[
  \frac{\sin^2 x}{1 - \sin^2 x} = t^2 \Rightarrow \sin^2 x = \frac{t^2}{1 + t^2}.
\]

√Оn fine, integrala devine:
\[
  I'(m) = 2m \int_0^\infty \frac{t^2}{(1 + m^2 t^2)(1 + t^2)} dt.
\]
FƒГc√Ѓnd descompunerea √Ѓn frac»Ыii simple, ob»Ыinem:
\[
  \frac{t^2}{(1 + m^2 t^2)(1 + t^2)} = \frac{1}{m^2 - 1} \Big( \frac{1}{t^2 + 1} - %
  \frac{1}{m^2 t^2 + 1} \Big).
\]
»Щi calculƒГm √Ѓn fine integrala $I'(m) = \frac{\pi}{m + 1}$. IntegrƒГm »Щi gƒГsim
$I(m) = \pi \ln(m + 1) + c$. Deoarece $I(1) = 0$, rezultƒГ $c = -\pi \ln 2$
»Щi, √Ѓn fine:
\[
  I(m) = \pi \ln \frac{m + 1}{2}.
\]

\vspace{0.5cm}

(b) DacƒГ considerƒГm func»Ыia:
\[
  f(\theta, a) = \ln(a^2 - \sin^2 \theta),
\]
observƒГm cƒГ este continuƒГ »Щi admite o derivatƒГ par»ЫialƒГ continuƒГ √Ѓn raport cu $a$.

Atunci avem:
\[

\]

Cu schimbarea de variabilƒГ $t = \tan \theta $, avem succesiv:
\begin{align*}
  I'(a) &= \int_0^\infty \frac{2a}{a^2 - \frac{t^2}{1 + t^2}} \cdot \frac{1}{1 + t^2} dt \\
        &= \int_0^\infty \frac{2a}{t^2(a^2 - 1) + a^2} dt \\
        &= \frac{2a}{a^2 - 1} \int_0^\infty \frac{1}{t^2 + \frac{a^2}{a^2 - 1}} dt \\
        &= \frac{2a}{a^2 - 1} \cdot \frac{\sqrt{a^2 - 1}}{a} \cdot \arctan t \frac{\sqrt{a^2 - 1}}{a} \Big|_0^\infty \\
        &= \frac{2}{\sqrt{a^2 - 1}} \cdot \frac{\pi}{2} \\
        &= \frac{\pi}{\sqrt{a^2 - 1}}.
\end{align*}

IntegrƒГm pentru a ob»Ыine $I(a)$ »Щi gƒГsim:
\[
  I(a) = \int \frac{\pi}{\sqrt{a^2 - 1}} da = \pi \ln(a + \sqrt{a^2 - 1}) + c.
\]

Pentru a calcula constanta $c$, putem rescrie integrala din forma ini»ЫialƒГ:
\begin{align*}
  I(a) &= \int_0^{\frac{\pi}{2}} \ln\Big( a^2\Big(1 - \frac{\sin^2 \theta}{a^2}\Big) \Big) d\theta \\
       &= \int_0^{\frac{\pi}{2}} \ln a^2 d\theta + \int_0^{\frac{\pi}{2}} \ln \Big(1 - \frac{\sin^2 \theta}{a^2} \Big) d\theta.
\end{align*}

Putem considera limita $ a \to \infty $ »Щi atunci integrala de calculat tinde la 0, deci $ c = -\pi \ln 2 $.

\vspace{0.5cm}

(c) DacƒГ considerƒГm func»Ыia $ f(x, a) = \dfrac{\arctan ax}{x \sqrt{1 - x^2}}$, observƒГm cƒГ este continuƒГ

Pentru a calcula integrala, facem schimbarea de variabilƒГ $ x = \sin t $ »Щi ob»Ыinem:

\end{align*}

A»Щadar, printr-o integrare √Ѓn func»Ыie de $ a $, gƒГsim:
