\chapter{Parțial 2018--2019}

\indent\indent 1. (a) Determinați natura seriei numerice:
\[
  \sum_{n \geq 1} \left( \sqrt{n + 2} - \sqrt{n + 1}\right) \cdot \sin \dfrac{1}{n}.
\]

(b) Studiați convergența simplă și uniformă a șirului de funcții:
\[
  f_n : (0, \infty) \to \RR, \quad f_n(x) = \arctan \dfrac{x}{n^2}, \quad n \geq 1.
\]

\vspace{0.5cm}

2. Determinați mulțimea de convergență și suma seriei de puteri:
\[
  \sum_{n \geq 0} (-1)^n \dfrac{1}{n!} \cdot \dfrac{x^{2n-1}}{5^n}.
\]

\vspace{0.5cm}

3. Folosind dezvoltarea în serie Taylor, să se calculeze $ \cos \dfrac{1}{2} $
cu 2 zecimale exacte.

\vspace{0.5cm}

4. Se consideră funcția $ \varphi : \RR^2 \to (0, \infty) $ de clasă $ \kal{C}^1 $
și funcția:
\[
  f(x, y) = xy \cdot \varphi \left( \ln(x + y) + \dfrac{x}{y} \right),
\]
cu $ x + y > 0 $.

Să se calculeze expresia:
\[
  E(x, y) = \dfrac{\partial f}{\partial x} + \dfrac{\partial f}{\partial y} - %
  \dfrac{x + y}{xy} \cdot f(x, y).
\]

\vspace{0.5cm}

5. Fie funcția: $ f : D \seq \RR^2 \to \RR, f(x, y) = x^2 + 2xy - 4x - y $.

Pentru $ D = \{ (x, y) \in \RR^2 \mid 0 \leq x \leq 1, 0 \leq y \leq -x^2 + 2x \} $,
determinați valoarea minimă și valoarea maximă a funcției.
