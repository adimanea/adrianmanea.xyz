\section{Ecua»Ыii de forma $ f(x, y^{(n)}) = 0 $}

      \[

\textbf{Exemplu:}
\[
  x - e^{y^{\prime\prime}} + y^{\prime\prime} = 0.

\emph{Solu»Ыie:} Putem nota $ y^{\prime\prime} = t $ »Щi atunci $ x(t) = e^t - t $.

\[

\]
Mai departe, $ dy = y^{\prime} dx $, deci:
\[
  dy = \Big( - \dfrac{t^2}{2} + te^t - e^t + c_1 \Big)(e^t - 1) dt.
\]
√Оn fine, solu»Ыia este:
\[
  y(t) = e^{2t} \Big( \dfrac{t}{2} - \dfrac{3}{4} \Big) + e^t \Big( - \dfrac{t^2}{2} + %
  1 + c_1 \Big) + \dfrac{t^3}{6} - c_1 t + c_2.
\]

\section{Ecua»Ыii de forma $ f(y^{(n-1)}, y^{(n)}) = 0 $}

Din nou, distingem douƒГ cazuri:
\begin{enumerate}[(a)]
  \item DacƒГ ecua»Ыia este rezolvabilƒГ prin func»Ыii elementare √Ѓn raport cu $ y^{(n)} $,
    atunci putem nota $ z = y^{(n-1)} $ »Щi avem $ z^{\prime} = f(z) $. Ecua»Ыia devine cu
    variabile separabile pentru $ z $ »Щi se rezolvƒГ corespunzƒГtor, conduc√Ѓnd la
  $ z = y^{(n-1)} = f_1(x, c_1) $, care este de tipul anterior;

    $ y^{(n)} $, dar cunoa»Щtem o reprezentare parametricƒГ de forma:

\emph{Solu»Ыie:} Facem schimbarea de func»Ыie $ z(x) = y^{\prime} $ »Щi ecua»Ыia devine:

De asemenea, trebuie sƒГ mai remarcƒГm »Щi solu»Ыiile particulare $ y = \pm x + c $.

\section{Ecua»Ыii de forma $ f(x, y^{(k)}, \dots, y^{(n)}) = 0 $}

Ecua»Ыia se rezolvƒГ cu schimbarea de func»Ыie $ y^{(k)} = z(x) $ »Щi rezultƒГ
o ecua»Ыie de ordin $ n - k $:
\[

\]

\textbf{Exemplu:} $ (1 + x^2)y^{\prime\prime} = 2xy^{\prime} $.

\emph{Solu»Ыie:} Cu schimbarea de func»Ыie $ y^{\prime} = z(x) $, ob»Ыinem ecua»Ыia:

\]

\[
  y(x) = c_1 x + c_1 \dfrac{x^3}{3} + c_2.
\]
\vspace{1cm}

\textbf{Exemplu 2:} $ y \cdot y^{\prime\prime} - yy^{^{\prime}2} = 0 $.

\emph{Solu»Ыie:} NotƒГm $ y^{\prime} = z $ »Щi ob»Ыinem:
\[
  y\cdot z^{\prime} - yz^2 = 0 \Rightarrow y(z^{\prime} - z^2) = 0,
\]
de unde rezultƒГ $ y = 0 $ sau $ z^{\prime} - z^2 = 0 $.

Din a doua variantƒГ, deducem succesiv:
\[
  \dfrac{dz}{dx} = z^2 \Rightarrow \dfrac{dz}{z^2} = dx \Rightarrow %
  -\dfrac{1}{z} = x + c_1.
\]
Mai departe:
\[
  -\dfrac{1}{y^{\prime}} = x + c_1 \Rightarrow y^{\prime} = \dfrac{-1}{x + c_1} %
  \Rightarrow y = -\ln(x + c_1) + c_2.
\]

\section{Ecua»Ыii de forma $ f(y^{\prime}, y^{\prime\prime}, \dots, y^{(n)}) = 0 $}

\section{Ecua»Ыii autonome (ce nu con»Ыin pe $ x $)}

\begin{remark}\label{rk:aut-const}
  ExistƒГ posibilitatea de a pierde solu»Ыii de forma $ y = c $ prin aceastƒГ
  metodƒГ, deci trebuie verificat dacƒГ este cazul separat.

\textbf{Exemplu:} $ 1 + y^{^{\prime}2} = 2yy^{\prime} $.
\emph{Solu»Ыie:} Putem lua $ y^{\prime} = p $ drept func»Ыie, iar pe $ y $ drept
variabilƒГ independentƒГ. RezultƒГ:

\]

\[
  \dfrac{2p dp}{1 + p^2} = \dfrac{dy}{y} \Rightarrow %

\]
Acum trebuie sƒГ ob»Ыinem pe $ x $ ca func»Ыie de $ p $ »Щi $ c_1 $.

$ dx = 2c_1 dp $. Deci $ x = 2c_1p + c_2 $, iar solu»Ыia generalƒГ devine
$ x(p) = 2c_1 p + c_2 $. Din expresia lui $ y $ de mai sus, rezultƒГ:
\[

\vspace{1cm}

\textbf{Exemplu 2:} $ y^{\prime\prime} = y^{^{\prime}2} = 2e^{-y} $.

\emph{Solu»Ыie:} NotƒГm $ y^{\prime} = p $ »Щi luƒГm ca necunoscutƒГ $ p = p(y) $.

\]

\[

\]
adicƒГ:
\[
  \dfrac{dy}{\pm \sqrt{c_1 e^{-2y} + 4e^{-y}}} = dx \Rightarrow %
  x + c_2 = \pm \dfrac{1}{2} \sqrt{c_1 + 4e^y}.
\]
Echivalent, putem scrie solu»Ыia »Щi √Ѓn forma implicitƒГ:
\[

\section{Ecua»Ыii Euler}

O ecua»Ыie Euler are forma generalƒГ:
\[
  \sum_{k = 0}^n a_k x^k y^{(k)} = f(x),
\]
pentru varianta omogenƒГ »Щi $ f(x) = 0 $ pentru varianta neomogenƒГ, cu $ a_i $
constante reale.

Ecua»Ыiile Euler se pot transforma √Ѓn ecua»Ыii cu coeficien»Ыi constan»Ыi prin nota»Ыia
(schimbarea de variabilƒГ) $ |x| = e^t $.

De remarcat faptul cƒГ, deoarece func»Ыia necunoscutƒГ este $ y = y(x) $, pentru
a ob»Ыine $ y^{\prime} = \dfrac{dy}{dx} $ trebuie sƒГ aplicƒГm regula de derivare
a func»Ыiilor compuse. Folosind nota»Ыia de la fizicƒГ $ \dot{y} = \dfrac{dy}{dt} $,
putem scrie:
\[
  \dfrac{dy}{dx} = \dfrac{dy}{dt} \cdot \dfrac{dt}{dx} = \dot{y} \cdot e^{-t}.
\]
Mai departe, de exemplu, pentru derivatele superioare:
\begin{align*}
  \dfrac{d^2 y}{dx^2} &= \dfrac{d}{dt} \Big( \dot{y} \cdot e^{-t} \Big) %
  \cdot \dfrac{dt}{dx} \\
                      &= e^{-2t} (\ddot{y} - \dot{y}).
\end{align*}

\vspace{1cm}

\textbf{Exemplu:} $ x^2 y^{\prime\prime} + xy^{\prime} + x = x $.

\emph{Solu»Ыie:} Facem substitu»Ыia $ |x| = e^t $ »Щi, folosind calculele
de mai sus, ob»Ыinem:
\[
  e^{2t} \cdot e^{-2t} (\ddot{y} - \dot{y}) - e^t \cdot e^{-t} + y(t) = e^t.
\]
Echivalent: $ \ddot{y} - 2\dot{y} + y = e^t $.

Aceasta este o ecua»Ыie liniarƒГ de ordinul al doilea, neomogenƒГ. Asociem ecua»Ыia
algebricƒГ $ r^2 - 2r + 1 = (r - 1)^2 = 0 $, deci solu»Ыia generalƒГ a variantei
omogene este:
\[
  y(t) = c_1 e^t + c_2 te^t.
\]

Folosind metoda varia»Ыiei constantelor, ob»Ыinem succesiv:
\begin{align*}
  y(t) &= c_1 e^t + c_2 t e^t + \dfrac{t^2 e^t}{2} \Rightarrow \\
  y(x) &= c_1 x + c_2 x \ln x + \dfrac{x \ln^2 x}{2}.
\end{align*}

\section{Exerci»Ыii}

\emph{Indica»Ыii:}

  \item Ecua»Ыia caracteristicƒГ este $ r^2 - r - 2 = (r + 1)(r - 2) = 0 $.
  \item Este o ecua»Ыie Euler √Ѓn raport cu $ t = x + 1 $.
