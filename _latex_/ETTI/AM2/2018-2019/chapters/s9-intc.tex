\index{integralƒГ!complexƒГ}

\section{Teorema reziduurilor}

Similar cu orice func»Ыie realƒГ, »Щi func»Ыiile complexe pot fi dezvoltate √Ѓn serii
de puteri. √Оn cazul complex, seriile se numesc \emph{serii Laurent} »Щi pot con»Ыine
»Щi puteri negative.

Informal, punctele cu probleme care ne intereseazƒГ se numesc \emph{poli}
sau \emph{puncte singulare}. Ordinul unui pol $ z = a $ este multiplicitatea algebricƒГ
a rƒГdƒГcinii $ z = a $ √Ѓn dezvoltarea √Ѓn serie Laurent a func»Ыiei $ f $. √Оn particular,
avem \emph{poli simpli, dubli} etc.

\begin{definition}\label{def:rez}\index{func»Ыii complexe!reziduu}
  Fie $ f : \CC \to \CC $ o func»Ыie complexƒГ »Щi dezvoltarea sa √Ѓn serie Laurent
  √Ѓn jurul unui punct $ z_0 \in \CC $:
  \[
    f(z) = \sum_{n \in \ZZ} a_n (z - z_0)^n, \quad a_n \in \RR.
  \]

  Se nume»Щte \emph{reziduul} func»Ыiei $ f $ √Ѓn punctul singular $ z_0 $ coeficientul
  $ a_{-1} $ din dezvoltarea de mai sus, notat $ \mathrm{Rez}(f, z_0) $.
\end{definition}

UrmƒГtoarea teoremƒГ ne dƒГ metode de calcul al reziduurilor, √Ѓn func»Ыie de multiplicitatea
lor:

\begin{theorem}[Calculul reziduurilor]\label{thm:calc-rez}\index{teorema!calculul reziduurilor}
  (1) $ \dr{Rez}(f, a) = c_{-1} $, unde $ c_{-1} $ este coeficientul lui $ \dfrac{1}{z-a} $
  √Ѓn dezvoltarea √Ѓn serie Laurent a func»Ыiei $ f $ √Ѓn vecinƒГtatea singularitƒГ»Ыii $ z = a $.

  (2) DacƒГ $ z = a $ este pol de ordinul $ p \geq 2 $ pentru $ f $, atunci:
  \[
    \dr{Rez}(f, a) = \dfrac{1}{(p-1)!} \lim_{z \to a} \Big[ (z - a)^p f(z) \Big]^{(p-1)};
  \]

  (3) DacƒГ $ z = a $ este pol simplu pentru $ f $, atunci, particulariz√Ѓnd formula
  de mai sus, avem:
  \[
    \dr{Rez}(f, a) = \lim_{z \to a} (z - a) f(z);
  \]

  (4) DacƒГ $ f $ se poate scrie ca un c√Ѓt de func»Ыii, $ f = \dfrac{A}{B} $, olomorfe √Ѓn
  jurul lui $ a $ »Щi dacƒГ $ z = a $ este pol simplu pentru $ f $, adicƒГ $ B(a) = 0 $,
  atunci:
  \[
    \dr{Rez}(f, a) = \lim_{z \to a} \dfrac{A(z)}{B'(z)}.
  \]
\end{theorem}

Rezultatul esen»Ыial al acestei sec»Ыiuni ne aratƒГ cƒГ, dacƒГ integrƒГm o func»Ыie
cu probleme, valoarea integralei este datƒГ √Ѓn mod esen»Ыial de reziduurile sale:

\begin{theorem}[Teorema reziduurilor]\label{thm:t-rez}\index{teorema!reziduurilor}
  Fie $ D \seq \CC $ un domeniu »Щi $ f : D - \{ \alpha_1, \dots, \alpha_k \} \to \CC $ o func»Ыie
  olomorfƒГ pentru care $ \alpha_i $ s√Ѓnt poli.

  Fie $ K \seq D $ un compact cu frontiera $ \Gamma = \partial K $, o curbƒГ de clasƒГ
  $ \kal{C}^1 $, jordanianƒГ, orientatƒГ pozitiv »Щi care con»Ыine toate $ \alpha_i $ √Ѓn
  interior. Atunci:
  \[
    \int_\Gamma f(z) dz = 2 \pi i \sum_{j = 1}^k \dr{Rez}(f, \alpha_j).
  \]
\end{theorem}

De exemplu, sƒГ calculƒГm integrala:
\[
  I = \int_{|z| = r} \frac{e^z}{(z - i)(z - 2)} dz, r > 0, r \neq 1, 2.
\]

\emph{Solu»Ыie:} DacƒГ $ 0 < r < 1 $, putem aplica teorema lui Cauchy (\ref{thm:tcauchy})
»Щi gƒГsim $ I = 0 $.

DacƒГ $ 1 < r < 2 $, aplicƒГm formula integralƒГ a lui Cauchy (\ref{thm:fintcauchy}) »Щi gƒГsim:
\[
  \int_{|z| = r} \frac{e^z}{(z - i)(z - 2)} dz = %
  \int_{|z| = r} \frac{\frac{e^z}{z - 2}}{z - i} dz = %
  \int_{|z| = r} \frac{f(z)}{z - i} dz,
\]
unde $ f(z) = \dfrac{e^z}{z - 2} $. RezultƒГ:
\[
  \int_{|z| = r} \frac{f(z)}{z - i} dz = 2 \pi i f(i) = 2 \pi i \frac{e^i}{i - 2}.
\]

DacƒГ $ r > 2 $, aplicƒГm teorema reziduurilor, cu $ i $ »Щi $ 2 $ poli simpli. Avem:
\begin{align*}
  \mathrm{Rez}(f, i) &= \lim_{z \to i} (z - i) \frac{e^z}{(z - i)(z - 2)} = \frac{e^i}{i - 2} \\
  \mathrm{Rez}(f, 2) &= \lim_{z \to 2} \frac{e^z}{(z - i)(z - 2)} = \frac{e^2}{2 - i}.
\end{align*}

RezultƒГ, din teorema reziduurilor:
\[
  \int_{|z| = r} \frac{e^z}{(z - i)(z - 2)} dz = 2 \pi i \Big( \frac{e^i}{i - 2} + %
  \frac{e^2}{2 - i} \Big).
\]

\section{Aplica»Ыii ale teoremei reziduurilor}

\indent\indent Putem folosi teorema reziduurilor pentru a calcula integrale trigonometrice de forma:
\[
  I = \int_0^{2 \pi} R(\cos \theta, \sin \theta) d \theta,
\]
unde $ R $ este o func»Ыie ra»ЫionalƒГ.

Facem schimbarea de variabilƒГ $ z = e^{i\theta} $ »Щi atunci, pentru $ \theta \in [0, 2\pi] $,
$ z $ descrie cercul $ |z| = 1 $, o datƒГ, √Ѓn sens direct.

Folosim formulele lui Euler:
\begin{align*}
  \cos \theta &= \frac{e^{i\theta} + e^{-i\theta}}{2} = \frac{1}{2} \Big( z + \frac{1}{z} \Big) \\
  \sin \theta &= \frac{e^{i\theta} - e^{-i\theta}}{2i} = \frac{1}{2i} \Big( z - \frac{1}{z} \Big).
\end{align*}

Atunci, dacƒГ $ z = e^{i\theta} $, rezultƒГ $ dz = ie^{i\theta}¬†d\theta = izd\theta $,

\[
  I = \int_0^{2 \pi} R(\cos \theta, \sin \theta) d\theta = \oint_{|z| = 1} R_1(z) dz,
\]
unde:
\[
  R_1(z) = \frac{1}{iz} R\Big( \frac{z^2 + 1}{2z}, \frac{z^2 - 1}{2iz} \Big).
\]

AceastƒГ func»Ыie poate avea poli »Щi deci putem folosi teorema reziduurilor. DacƒГ
$ a_1, \dots, a_n $ s√Ѓnt polii din interiorul cercului unitate, avem:
\[

\]

\vspace{1cm}

SƒГ vedem c√Ѓteva exemple:
\begin{enumerate}[(a)]
\item $ \ds\int_0^{2¬†\pi} \frac{d\theta}{2 + \cos \theta} $;
\item $ \ds\int_0^{2¬†\pi} \frac{d\theta}{1 + 3 \cos^2 \theta} $;
\item $ \ds\int_0^{2 \pi} \frac{\sin \theta}{1 + i \sin \theta}¬†d\theta $;
\item $ \ds\int_0^{2 \pi} \frac{d\theta}{1 + a \sin \theta}, |a| < 1, a \in \RR $.
\end{enumerate}

\emph{Solu»Ыie:}

(a) NotƒГm $ z = e^{i\theta} $, cu $ \theta \in [0, 2\pi] $. Atunci avem succesiv:
\begin{align*}
  dz &= ie^{i\theta} d\theta = izdz \Rightarrow d\theta = \frac{dz}{iz} \\
  \cos \theta &= \frac{e^{i\theta} + e^{-i\theta}}{2} = \frac{1}{2} \Big( z + \frac{1}{z} \Big) \\
  \frac{1}{2 + \cos \theta} &= \frac{2z}{z^2 + 4z + 1} \\
  \int_0^{2 \pi} \frac{d\theta}{2 + \cos \theta} &= \oint_{|z| = 1} \frac{2z}{z^2 + 4z + 1} \frac{dz}{iz} \\
     &= -2i \oint_{|z| = 1} \frac{dz}{z^2 + 4z + 1}.
\end{align*}

Acum folosim teorema reziduurilor. SingularitƒГ»Ыile func»Ыiei $ f(z) = \dfrac{1}{z^2 + 4z + 1} $
s√Ѓnt $ z = -2 \pm \sqrt{3}¬†$, care s√Ѓnt poli simpli. Numai $ z = -2 + \sqrt{3} $ se aflƒГ
√Ѓn interiorul cercului $ |z| = 1 $ »Щi calculƒГm reziduul folosind
Teorema \ref{thm:calc-rez}(2).
